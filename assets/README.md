# Assets Folder

This folder contains the app's visual assets. The current files are placeholders.

## Required Assets

Please replace the placeholder files with actual images:

1. **icon.png** - App icon (1024x1024 px)
   - Should be a square image
   - Represents your app on the home screen

2. **splash.png** - Splash screen image
   - Displayed when the app is loading
   - Recommended size: 1284x2778 px

3. **adaptive-icon.png** - Android adaptive icon (1024x1024 px)
   - Used for Android devices
   - Should have a transparent background

4. **favicon.png** - Web favicon (48x48 px or larger)
   - Used when running the app in a web browser

## Creating Your Assets

You can use design tools like:
- Figma
- Adobe Illustrator
- Canva
- Online icon generators

For the Study Quest theme, consider using:
- Book or graduation cap icons
- The app's primary color: #6366f1 (indigo)
- Clean, modern design aesthetic

## Note

Until you add proper images, you may see warnings when running the app. The app will still function correctly, but the visual assets will be missing.
